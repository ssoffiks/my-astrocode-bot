# Мой Астрокод — Telegram bot

Кодовая версия бота вместо одного AutonomousNode/Botpress-узла.

## Что уже есть

- анкета мини-профиля;
- сохранение данных пользователя в SQLite;
- команды `/start`, `/profile`, `/compatibility`, `/meditation`, `/moon`, `/channel`, `/help`, `/terms`, `/privacy`, `/support`, `/paysupport`;
- редактирование и удаление профиля;
- базовая совместимость по двум датам;
- медитация дня;
- бесплатный расчёт знака Луны по дате, времени и городу рождения;
- продающие страницы;
- Telegram Stars invoices для цифровых продуктов;
- сохранение успешных покупок и `telegram_payment_charge_id`.
- безопасная read-only админка для статистики и просмотра покупок.

## Как запустить локально

1. Установи Node.js 22 или новее.
2. Скопируй `.env.example` в `.env`.
3. Вставь в `.env` токен бота из BotFather:

```env
BOT_TOKEN=твой_токен
SUPPORT_USERNAME=@твой_ник
ADMIN_IDS=
```

4. Установи зависимости:

```bash
npm install
```

5. Проверь тесты:

```bash
npm test
```

6. Запусти бота:

```bash
npm start
```

## Важно про оплату

Цифровые продукты внутри Telegram продаём через Telegram Stars, валюта `XTR`.
Карточную оплату внутри бота не подключаем как отдельный способ за цифровой продукт: вместо этого пользователь покупает Stars картой в Telegram и возвращается к оплате в боте.

Перед отправкой Stars invoice бот показывает короткое подтверждение условий: пользователь может открыть `/terms`, `/privacy`, вернуться в меню или согласиться и перейти к оплате. После подтверждения факт согласия сохраняется в SQLite с версиями условий и политики приватности.

## Бесплатный расчёт Луны

Раздел «🌙 Узнать свою Луну» доступен из главного меню и по команде `/moon`.

Бот спрашивает:

- дату рождения;
- время рождения или вариант «Не знаю время»;
- город рождения.

Если город есть в минимальной базе часовых поясов, бот сам подставит нужный часовой пояс. Если город не найден, он попросит UTC-смещение вручную, например `+3`, `+03:00` или `-05:00`.

Если время рождения неизвестно, расчёт делается на 12:00 местного времени. Бот дополнительно проверяет знак Луны на начало и конец дня и предупреждает, если в этот день Луна могла перейти в другой знак.

Результат сохраняется в SQLite в таблице `moon_profiles`. Отдельную миграцию запускать не нужно: таблица создаётся автоматически при `initDb()`.

## Админка

Админка работает только для Telegram ID, перечисленных в переменной окружения `ADMIN_IDS`.

1. Напиши боту команду `/myid`.
2. Бот ответит твоим Telegram ID.
3. Добавь ID в `.env`:

```env
ADMIN_IDS=123456789
```

Если администраторов несколько, укажи их через запятую:

```env
ADMIN_IDS=123456789,987654321
```

Пробелы вокруг ID не мешают. Если `ADMIN_IDS` пустой, админка недоступна никому.

Команды админки:

```text
admin - меню админки
admin_help - помощь по админке
admin_stats - общая статистика
admin_recent - последние 10 покупок
admin_recent 20 - последние 20 покупок, максимум 50
admin_user 123456789 - профиль и покупки пользователя
```

Рассылки специально не добавлены: это отдельный опасный функционал, его лучше делать позже с предпросмотром и подтверждением.

После изменения `ADMIN_IDS` на VPS нужно перезапустить процесс:

```bash
pm2 restart my-astrocode-bot
pm2 save
```
