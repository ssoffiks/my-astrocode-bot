# Мой Астрокод — Telegram bot

Кодовая версия бота вместо одного AutonomousNode/Botpress-узла.

## Что уже есть

- анкета мини-профиля;
- сохранение данных пользователя в SQLite;
- команды `/start`, `/profile`, `/compatibility`, `/meditation`, `/help`, `/terms`, `/privacy`, `/support`, `/paysupport`;
- редактирование и удаление профиля;
- базовая совместимость по двум датам;
- медитация дня;
- продающие страницы;
- Telegram Stars invoices для цифровых продуктов;
- сохранение успешных покупок и `telegram_payment_charge_id`.

## Как запустить локально

1. Установи Node.js 22 или новее.
2. Скопируй `.env.example` в `.env`.
3. Вставь в `.env` токен бота из BotFather:

```env
BOT_TOKEN=твой_токен
SUPPORT_USERNAME=@твой_ник
```

4. Установи зависимости:

```bash
npm install
```

5. Проверь тесты:

```bash
npm test
```

6. Запусти бота:

```bash
npm start
```

## Команды для BotFather

```text
start - собрать мини-астропрофиль
profile - мой мини-профиль
compatibility - совместимость
meditation - медитация дня
help - меню и помощь
paysupport - поддержка по оплатам
support - поддержка
terms - условия использования
privacy - политика приватности
```

## Важно про оплату

Цифровые продукты внутри Telegram продаём через Telegram Stars, валюта `XTR`.
Карточную оплату внутри бота не подключаем как отдельный способ за цифровой продукт: вместо этого пользователь покупает Stars картой в Telegram и возвращается к оплате в боте.

Перед отправкой Stars invoice бот показывает короткое подтверждение условий: пользователь может открыть `/terms`, `/privacy`, вернуться в меню или согласиться и перейти к оплате. После подтверждения факт согласия сохраняется в SQLite с версиями условий и политики приватности.
